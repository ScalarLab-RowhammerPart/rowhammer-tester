DFI_SWITCH_REFRESH_COUNT¶

Address: 0x82007800 + 0x0 = 0x82007800

Count of all refresh commands issued (both by Memory Controller and Payload Executor). Value is latched from internal counter on mode trasition: MC -> PE or by writing to the refresh_update CSR.

DFI_SWITCH_REFRESH_UPDATE¶

Address: 0x82007800 + 0x4 = 0x82007804

Force an update of the refresh_count CSR.
